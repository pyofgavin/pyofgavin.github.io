random的使用

random.random()用于生成一个0到1的随机浮点数：0<=n<1.0（左闭右开）

random.uniform(a,b) # 用于生成一个指定范围的随机浮点数,两个参数其中一个是上限,一个是下限,如果a>b,则生成的随机数n:a<=n<b。如果a<b,则b<=n<a

random.randint(a,b) # 用于生成一个指定范围内的整数,其中参数a是下限,参数b是上限,生成的随机数n:a<=n<b

random.choice(sequence) # random.choice从序列中获取一个随机元素,
random.shuffle(x[,random]) # 用于将一个列表中的元素打乱,即将列表内的元素随机排序

random.sample(sequence,k) # 从指定序列中随机获取指定长度的片断并随机排序。

多进程编程

什么是并发编程？

  • 并发编程是一种充分发挥硬件性能的多任务程序设计模式，单核CPU环境下一个时间段上会有多个程序并行执行，但是在同一个时间点上只允许执行一个进程，并发编程有三种形式多进程编程、多线程编程、多协程编程

什么是进程

  • 进程（Process）是系统进行资源分配的调度运行的基本单位，一个进程组成有三部分程序、数据、进程控制块 PCB
      缩进的快速实现，全角空格+ 配合互补：正常输入时全角空格最快捷，列表后用两个 缩进，之后便可恢复全角空格方式。
  • 进程是一个动态的实体：
            1. 创建状态：分配一个PCB
        2. 就绪状态：除CPU全部资源，等待CPU调度
        3. 执行状态：获得CPU，开始执行
        4.  阻塞状态：暂停
        5.  终止状态
  • 早起的磁盘操作系统（Disk Operating System）简称DOS系统，采用单进程处理模式，同一个时间段系统所有你资源只为一个程序服务。
  • Python多进程编程可以通过 multiprocessing模块实现，在该模块中有专门的处理类 Process
  • multiprocessing.current_process()函数动态获取当前正在执行的进程对象

类和对象

面向对象是现在最为流行的软件设计与开发方法。

面向过程与面向对象的区别

面向过程： 由个人准备好所有制造机器的材料，然后按照自己的标准制造机器人，这样制造的机器人不具备通用性。
面向对象：由设计团队将机器人的制造工艺进行拆分，而后详细设计出每一个部件的定义标准抑菌剂各个部件之间的连接标准，随后将这些设计标准交付响应的工厂进行制造，每个工厂的流水线只能完成一个零部件的生产，最后再一起进行拼接。

面向对象编程的三大主要特点

  • 封装性
  • 继承性
  • 多态性

    类与对象

面向对象编程中类和对象是基本的组成单元，类是对一个客观群体特征的抽象描述，而对象表示的一个个具体的可操作实物。
类是对象的模板 而对象是类的实例
