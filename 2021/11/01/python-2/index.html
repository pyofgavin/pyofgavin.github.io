python语言的优点

    python 可以做什么

  • 网站后端程序员：使用python搭建网站，例如Gmail、Youtube、Reddit、知乎、豆瓣
  • 自动化运维
  • 数据分析师
  • 游戏开发
  • 自动化测试
  • 网站的开发 Django flask bootstrap
  • 用爬虫爬取或处理大量信息 批量下载枚举、爬合适房源、刷折扣机票、运行投资策略

    变量与字符串

  • 变量是编程中最基本的存储单位
  • 变量的名称叫标识符
  • 给变量a赋值42 a=42
  • print()打印函数
  • type（）查看类型函数
  • 数据类型有：int、float、str、dict、set、list、、tuple、bool
  • len（）计算长度函数

    字符串的切片与索引###

  • string=’helloworld’
  • string[1:5]=trin 原则：左闭右开原则
  • replace（old，new）
  • find(str，开始索引，结束索引) 返回开始的索引值

字符串的格式化输出

print('{} a word she can get what she {} for.'.format('with','came'))
print('{0} a word she can get what she {1} for.'.format('with','came'))

函数

  • def（define） 创建一个函数

  • arg (argument) 参数

  • return 返回结果

  • insert(1，apple) 插入

  • extend（列表）为一个列表追加另外一个列表

  • append（data）添加单个元素

  • clear()清楚列表数据

  • copy()列表复制

  • count(data)统计某一个数据在列表中出现的次数

  • index(data) 从列表中查询某个数据第一次出现的位置

  • reverse()反转

  • sort()列表数据排序

  • remove(data)从列表删除数据

  • pop(index)从列表弹出一个数据删除

  • id()获取列表地址信息

  • split(‘,’)将字符串用’,’分开

  • strip()

    字典的特征###

  • 字典中的数据是以键值对的形式出现的：key value

  • 键是不能重复，而值可以

  • 字典中的key是不能修改的，而值可以改变，可以是任何对象

  • 字典在索引内容的时候跟列表一样，例如 dict[key]
    ###集合的特征 ###

  • 每个集合中的元素是无序的、不重复的任意对象

  • 利用集合可以把数据结构中重复的元素减掉

  • a_set={1,2,3,4,5}

  • a_set.add(6) 添加元素

  • a_set.discard(6) 删除元素
